Indegene Task:

 Build a rest GET API endpoint using Django that would pull data from the Excel provided in the Technical_assessment_source_data.xlsx ,transform the data into a JSON response provided in the JSON file assignment_response.json.Feel free make any necessary assumptions and specify them in the Readme.Ignore if any extra key values are there in assignment_response.json.
